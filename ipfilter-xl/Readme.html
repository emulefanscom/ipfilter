迅雷与旋风离线及资源服务器IP过滤列表使用说明

by eMuleFans.com

eMule、aMule使用ipfilter.dat

导入

针对eMule以及其Mod用户的ipfilter.dat导入方法（aMule大概也类似）：

下载并解压出ipfilter.dat文件；打开eMule的“工具”（Tools）->“IP过滤…”（IP Filter…），点击“附加…”（Append…）按钮，找到并打开解压好的迅雷服务器ipfilter.dat文件，即可在你原来存在的ipfilter基础上导入此迅雷服务器IP列表。

直接使用

如果你只想屏蔽迅雷和旋风离线而其他IP地址都不需要屏蔽的话，那么可以直接将离线IP过滤列表用于eMule：

把解压出来的文件拷贝到config目录里（config目录：Win XP，eMule的config目录在eMule安装目录下；Win Vista、Win 7以上，config目录默认在C:\Users\你的用户名\AppData\Local\eMule\config\，这取决于你的设置，详细请见这里），然后启动eMule即可。

或者，你也可以打开eMule的“选项”=>“安全”窗口，粘贴http://ed2k.im/ipxl地址（其指向直接的.dat），点击“加载”按钮即可下载并加载最新离线IP过滤列表。

其他电驴、BT软件使用ipfilter.dat

可以使用IPFilter的不仅仅只有eMule（或其Mod）一种，aMule、MLDonkey等许多电驴软件，和μTorrent、Vuze、BitTorrent、BitSpirit（比特精灵）、BitComet（比特彗星）、Halite等各大主流BitTorrent协议软件都可以使用IPFilter。

如想在原有基础上导入本IP过滤列表，请参见各个软件的帮助文档；也可以用文本编辑器打开你原有的，和这里下载了的ipfilter.dat，在前者的末尾换一行后，将后者复制粘贴到前者末尾，即可（因为一般ipfilter.dat中出现重复的IP没有关系，程序都能识别，有的运行时也能自动转换为不重复的正常IP列表）。

如想直接使用这里的过滤列表，那么直接把ipfilter.dat放到相应文件夹下即可。具体文件夹位置见下。

上述软件均可兼容使用不分级的、ipfilter.dat格式的IPFilter（但有的不支持“#”号开头的行注释，我们这里提供的ipfilter.dat已不再有行注释）。具体每个软件ipfilter.dat放置的位置列举如下（仅供参考，一般还需要视你的操作系统、Win用户名（“Administrator”或其他）、软件安装位置（C:\Program Files\???或其他位置）、软件版本等而定。鼠标悬停在下文中虚下划线文字上看注释）：

  • eMule：C:\Program Files\eMule\config\（WinXP）或C:\Users\Administrator\AppData\Local\eMule\config\（Vista、Win7以上默认，但可以设置）
  • aMule：~/.aMule/ipfilter.dat
  • MLDonkey：在设置中将ip_blocking设置为IP过滤列表文件的位置，或用block_list命令加载文件
  • μTorrent：C:\Users\Administrator\AppData\Roaming\uTorrent\
  • Vuze：C:\Program Files\Vuze\
  • BitTorrent：C:\Users\Administrator\AppData\Roaming\BitTorrent\
  • BitSpirit：C:\Program Files\BitSpirit\config\
  • BitComet：C:\Program Files\BitComet\rules\
  • Halite：C:\Program Files\Halite\

Apache的.htaccess

.htaccess是Apache HTTP 服务器的配置文件。打开你的站点顶层目录下的.htaccess，将这里的.htaccess中的内容复制粘贴到你的.htaccess的最后；如果你的站点顶层目录下没有.htaccess，那么可以直接把这里的.htaccess放在该目录下。这样即可阻止列表中的迅雷资源服务器和离线IP访问、搜集、下载你的网站上的所有文件。

guarding.p2p

PeerGuardian和PeerBlock等专业IP过滤程序的guarding.p2p格式IP过滤列表，MLDonkey等部分电驴与BT软件也兼容。如果要用，请参考这些软件的文档。

迅雷與旋風離線及資源伺服器IP過濾列表使用說明

by eMuleFans.com

eMule、aMule使用ipfilter.dat

導入

針對eMule以及其Mod使用者的ipfilter.dat導入方法（aMule大概也類似）：

下載並解壓出ipfilter.dat檔案；打開eMule的「工具」（Tools）->「IP過濾…」（IP Filter…），點擊「附加…」（Append…）按鈕，找到並打開解壓好的迅雷伺服器ipfilter.dat檔案，即可在你原來存在的ipfilter基礎上導入此迅雷伺服器IP列表。

直接使用

如果你只想禁止迅雷和旋風離線而其他IP地址都不需要禁止的話，那麼可以直接將離線IP過濾列表用於eMule：

把解壓出來的檔案拷貝到config目錄里（config目錄：Win XP，eMule的config目錄在eMule安裝目錄下；Win Vista、Win 7以上，config目錄預設在C:\Users\你的使用者名稱\AppData\Local\eMule\config\，這取決於你的設定，詳細請見這裡），然後啟動eMule即可。

或者，你也可以打開eMule的「選項」=>「安全」視窗，粘貼http://ed2k.im/ipxl地址（其指向直接的.dat），點擊「載入」按鈕即可下載並載入最新離線IP過濾列表。

其他電驢、BT軟體使用ipfilter.dat

可以使用IPFilter的不僅僅只有eMule（或其Mod）一種，aMule、MLDonkey等許多電驢軟體，和μTorrent、Vuze、BitTorrent、BitSpirit（比特精靈）、BitComet（比特彗星）、Halite等各大主流BitTorrent協定軟體都可以使用IPFilter。

如想在原有基礎上導入本IP過濾列表，請參見各個軟體的幫助文檔；也可以用純文字檔案編輯器打開你原有的，和這裡下載了的ipfilter.dat，在前者的末尾換一行后，將後者複製粘貼到前者末尾，即可（因為一般ipfilter.dat中出現重複的IP沒有關係，程式都能識別，有的運行時也能自動轉換為不重複的正常IP列表）。

如想直接使用這裡的過濾列表，那麼直接把ipfilter.dat放到相應資料夾下即可。具體資料夾位置見下。

上述軟體均可兼容使用不分級的、ipfilter.dat格式的IPFilter（但有的不支援「#」號開頭的行註解，我們這裡提供的ipfilter.dat已不再有行註解）。具體每個軟體ipfilter.dat放置的位置列舉如下（僅供參考，一般還需要視你的作業系統、Win使用者名稱（「Administrator」或其他）、軟體安裝位置（C:\Program Files\???或其他位置）、軟體版本等而定。滑鼠懸停在下文中虛下劃線文字上看註解）：

  • eMule：C:\Program Files\eMule\config\（WinXP）或C:\Users\Administrator\AppData\Local\eMule\config\（Vista、Win7以上預設，但可以設定）
  • aMule：~/.aMule/ipfilter.dat
  • MLDonkey：在設定中將ip_blocking設定為IP過濾列表檔案的位置，或用block_list命令載入檔案
  • μTorrent：C:\Users\Administrator\AppData\Roaming\uTorrent\
  • Vuze：C:\Program Files\Vuze\
  • BitTorrent：C:\Users\Administrator\AppData\Roaming\BitTorrent\
  • BitSpirit：C:\Program Files\BitSpirit\config\
  • BitComet：C:\Program Files\BitComet\rules\
  • Halite：C:\Program Files\Halite\

Apache的.htaccess

.htaccess是Apache HTTP 伺服器的配置檔案。打開你的站點頂層目錄下的.htaccess，將這裡的.htaccess中的內容複製粘貼到你的.htaccess的最後；如果你的站點頂層目錄下沒有.htaccess，那麼可以直接把這裡的.htaccess放在該目錄下。這樣即可阻止列表中的迅雷資源伺服器和離線IP訪問、搜集、下載你的網站上的所有檔案。

guarding.p2p

PeerGuardian和PeerBlock等專業IP過濾程式的guarding.p2p格式IP過濾列表，MLDonkey等部分電驢與BT軟體也兼容。如果要用，請參考這些軟體的文檔。
