迅雷与旋风离线及资源服务器IP过滤列表使用说明

by eMuleFans.com

eMule、aMule使用ipfilter.dat

导入

针对eMule以及其Mod用户的ipfilter.dat导入方法（aMule大概也类似）： 下载并解压出ipfilter.dat文件；打开eMule的“工具”（Tools）->“IP过滤…”（IP Filter…），点击“附加…”（Append…）按钮，找到并打开解压好的迅雷服务器ipfilter.dat文件，即可在你原来存在的ipfilter基础上导入此迅雷服务器IP列表。

直接使用

如果你只想屏蔽迅雷和旋风离线而其他IP地址都不需要屏蔽的话，那么可以直接将离线IP过滤列表用于eMule： 把解压出来的文件拷贝到config目录里（config目录：Win XP，eMule的config目录在eMule安装目录下；Win Vista、Win 7以上，config目录默认在C:\Users\你的用户名\AppData\Local\eMule\config\，这取决于你的设置，详细请见这里），然后启动eMule即可。 或者，你也可以打开eMule的“选项”=>“安全”窗口，粘贴http://ed2k.im/ipxl地址（其指向直接的.dat），点击“加载”按钮即可下载并加载最新离线IP过滤列表。

其他电驴、BT软件使用ipfilter.dat

可以使用IPFilter的不仅仅只有eMule（或其Mod）一种，aMule、MLDonkey等许多电驴软件，和μTorrent、Vuze、BitTorrent、BitSpirit（比特精灵）、BitComet（比特彗星）、Halite等各大主流BitTorrent协议软件都可以使用IPFilter。 如想在原有基础上导入本IP过滤列表，请参见各个软件的帮助文档；也可以用文本编辑器打开你原有的，和这里下载了的ipfilter.dat，在前者的末尾换一行后，将后者复制粘贴到前者末尾，即可（因为一般ipfilter.dat中出现重复的IP没有关系，程序都能识别，有的运行时也能自动转换为不重复的正常IP列表）。 如想直接使用这里的过滤列表，那么直接把ipfilter.dat放到相应文件夹下即可。具体文件夹位置见下。 上述软件均可兼容使用不分级的、ipfilter.dat格式的IPFilter（但有的不支持“#”号开头的行注释，我们这里提供的ipfilter.dat已不再有行注释）。具体每个软件ipfilter.dat放置的位置列举如下（仅供参考，一般还需要视你的操作系统、Win用户名（“Administrator”或其他）、软件安装位置（C:\Program Files\???或其他位置）、软件版本等而定。鼠标悬停在下文中虚下划线文字上看注释）：
  • eMule：C:\Program Files\eMule\config\（WinXP）或C:\Users\Administrator\AppData\Local\eMule\config\（Vista、Win7以上默认，但可以设置）
  • aMule：~/.aMule/ipfilter.dat
  • MLDonkey：在设置中将ip_blocking设置为IP过滤列表文件的位置，或用block_list命令加载文件
  • μTorrent：C:\Users\Administrator\AppData\Roaming\uTorrent\
  • Vuze：C:\Program Files\Vuze\
  • BitTorrent：C:\Users\Administrator\AppData\Roaming\BitTorrent\
  • BitSpirit：C:\Program Files\BitSpirit\config\
  • BitComet：C:\Program Files\BitComet\rules\
  • Halite：C:\Program Files\Halite\

Apache的.htaccess

.htaccess是Apache HTTP 服务器的配置文件。打开你的站点顶层目录下的.htaccess，将这里的.htaccess中的内容复制粘贴到你的.htaccess的最后；如果你的站点顶层目录下没有.htaccess，那么可以直接把这里的.htaccess放在该目录下。这样即可阻止列表中的迅雷资源服务器和离线IP访问、搜集、下载你的网站上的所有文件。

guarding.p2p

PeerGuardian和PeerBlock等专业IP过滤程序的guarding.p2p格式IP过滤列表，MLDonkey等部分电驴与BT软件也兼容。如果要用，请参考这些软件的文档。
